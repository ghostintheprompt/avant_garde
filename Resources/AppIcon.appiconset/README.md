# Avant Garde App Icon

## Design Concept

The Avant Garde app icon should embody:
- **Professional authoring** - Clean, sophisticated design
- **Color psychology** - Gradient or vibrant colors representing creativity
- **Modern aesthetics** - macOS Big Sur style with depth and dimension

## Recommended Design Elements

### Icon Concept 1: Book + Palette
- A stylized book or document
- Color palette/brush stroke overlay
- Gradient from blue (focus) to purple (creativity)

### Icon Concept 2: Pen + Audio Wave
- Elegant fountain pen or quill
- Audio waveform integrated into design
- Professional gradient (dark purple to vibrant orange)

### Icon Concept 3: Abstract "AG" Monogram
- Stylized "AG" letters
- Modern, geometric design
- Colorful gradient background

## Required Sizes

Generate PNG files at these exact dimensions:
- 16x16, 32x32 (retina @2x)
- 32x32, 64x64 (retina @2x)
- 128x128, 256x256 (retina @2x)
- 256x256, 512x512 (retina @2x)
- 512x512, 1024x1024 (retina @2x)

## App Store Requirements

- 1024x1024 PNG for App Store listing (no alpha channel)
- All icons should have consistent visual style
- Avoid text in icons (it becomes unreadable at small sizes)
- Use simple, recognizable shapes
- Ensure good contrast for both light and dark modes

## Tips for macOS Icons

1. **Use depth** - Subtle shadows and highlights
2. **Round corners** - macOS icons have rounded corners
3. **Grid alignment** - Follow Apple's icon grid template
4. **Test at all sizes** - Icons should work at 16x16
5. **Avoid gradients that are too complex** - Keep it simple
