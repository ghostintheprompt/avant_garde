#### Editor Features
- Rich text editing with full formatting support
- Chapter navigator with drag-and-drop organization
- Real-time word count and statistics
- Character count and reading time estimates
- Professional formatting toolbar
- Image insertion and management
- Footnote support
- Chapter break insertion
- Multiple alignment options (left, center, right)
- Font customization

#### Audio Features
- High-quality text-to-speech integration
- Chapter-by-chapter audio playback
- Voice selection (Samantha, Alex, Ava, Tom)
- Adjustable playback speed, pitch, and volume
- Audio controls in menu and toolbar
- Voice setup guide for downloading premium voices

#### Export & Publishing
- One-click KDP HTML export
- One-click Google Play Books EPUB export
- Real-time format validation
- Platform-specific optimization
- Export progress indicators
- Format compatibility checking

#### User Interface
- Beautiful welcome screen with onboarding tutorial

- Support for system dark mode
- Retina display optimization

### Technical

#### Architecture
- Swift 5.7+ codebase
- AppKit-based native macOS app
- Model-View-Controller architecture
- Modular component design
- Clean separation of concerns

#### Performance
- Optimized for large documents
- Efficient memory usage
- Fast theme switching
- Responsive UI even with long texts
- Background processing for exports

#### Compatibility
- macOS 12.0+ (Monterey and later)
- Intel and Apple Silicon support
- High DPI display support
- Multiple monitor support

### Security & Privacy
- No data collection
- All processing done locally
