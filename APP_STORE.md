## Review Notes for Apple

**Testing Account**: Not required (no login system)

**Demo Content**: The app includes a demo mode that showcases all features without requiring user content.

**Special Instructions**:
1. The app will show a welcome screen on first launch - this is expected behavior
2. Text-to-speech requires macOS system voices (already included in macOS)
3. Export features generate HTML and EPUB files - no external services required
4. All data is stored locally on the Mac

**Features to Test**:
- Color theme switching (View ‚Üí Color Themes menu)
